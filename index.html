MOTOR FLAMINGO

Flamingo es un motor diseñado para la creación de videojuegos en 3D. Está desarrollado con C++ y usa bibliotecas como Ogre para el renderizado gráfico, Bullet para la física, LUA para scripting y SDL entre otros. A partir de este motor fue desarrollado el juego Eldersbane. Este es un pequeño juego de acción y aventuras en el que tendremos que explorar el mundo derrotando a enemigos.

Ver en GitHub

MASTERMIND

Juego Mastermind desarrollado para dispositivos móviles en Android Studio. Este presenta un sistema de guardado de las partidas y la posible personalización de los elementos del juego. Además se hace uso del acelerómetro del dispositivo móvil para dotarlo de una mayor jugabilidad y el uso de anuncios como posible sistema de monetizado y notificaciones para informar al jugador.

Ver en GitHub

SISTEMA DE DETECCIÓN DE ZONAS NO ACCESIBLES EN MAPAS 3D

Desarrollo de un sistema para comprobar la accesibilidad de las zonas de un mapa 3D a través de bots con movimiento automático desarrollado en Unity. En ella se busca comprobar la transitabilidad de terrenos o mapas de videojuegos comprobando el porcentaje de cobertura, estimando la navegabilidad de los mapas creando para ello mapas de calor.

Ver en GitHub

MONKEY DELIVERY

Monkey Delivery es un juego desarrollado en C++ a través de la biblioteca de SDL. En él somos un pequeño mono que tendrá que ir completando repartos a los diferentes vecinos de una isla.

Ver en GitHub

BIRD SEASON

Bird Season es un juego creado en Unity haciendo uso de maquinas de estados para simular un ciclo de vida de un gorrión. En el juego tendremos que ir manipulando el clima a través de comandos de voz para crear efectos en las estaciones. Todo el audio del juego esta creado a partir de FMOD.

Ver en GitHub

BATTLE SHIP EN LINEA

Battle Ship es un juego en línea de naves para dos jugadores desarrollado en C para sistemas Linux. En este juego tendremos que luchar contra el otro jugador hasta que solo uno de los dos quede en pie.

LA GAMIFICACIÓN COMO HERRAMIENTA PARA FOMENTAR Y MOTIVAR LA ACTIVIDAD FÍSICA

Aplicación móvil que fomenta la actividad física mediante gamificación, sensores y geoposicionamiento. Utiliza MongoDB y Spring Boot para la gestión de datos, y hardware externo como la pulsera EmotiBit para recopilar señales ECG y GSR, evaluando su impacto en el tiempo de ejercicio y la condición física de los usuarios.

Ver en GitHub Ver publicación

JUEGO MULTIPLATAFORMA PARA PS5 Y PC

Desarrollo de un pequeño motor para la creación de un juego compatible tanto para PC como para PS5. Desarrollo en PS5 a través de un contrato de confidecialidad. El juego desarrollado es una replica de "La Plaga Galáctica", publicado en 1984 y desarrollado por Paco Suárez.
